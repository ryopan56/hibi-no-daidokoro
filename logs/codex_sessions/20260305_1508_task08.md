# Task08 セッションログ

## 指示内容
- `/calendar`（実装は `calendar/`）で月表示カレンダーを追加。
- ログがある日だけ控えめな印を表示。
- 日付クリックで既存の `meallog_detail` へ遷移。
- 対象はログイン中ユーザーのデータのみ。
- 不正な `year`/`month` は当月へフォールバック。

## Codexの応答
- `meallogs.views` に月カレンダー表示ロジックを追加。
- `src/hibi_no_daidokoro/urls.py` に `calendar/`（name: `meallog_calendar`）を追加。
- `meallogs/calendar.html` を `base.html` 拡張テンプレートとして新規追加。
- カレンダー用テストを `src/meallogs/tests.py` に追加。
- `check` と `test` を Docker 上で実行し成功を確認。

## 実施変更
- `src/meallogs/views.py`: 年月パース、月グリッド生成、ログ有無フラグ、前月/翌月導線を実装。
- `src/hibi_no_daidokoro/urls.py`: `calendar/` ルート追加（スラ無しは未追加）。
- `src/meallogs/templates/base.html`: 共通レイアウトの最小テンプレートを追加。
- `src/meallogs/templates/meallogs/calendar.html`: `extends` ベースのカレンダー画面を追加。
- `src/meallogs/tests.py`: ログイン必須、自分のログのみ印表示、不正年月フォールバックのテストを追加。

## 実行コマンド
- `git status -sb`
- `git checkout -b feat/task-08-calendar`
- `docker compose exec web python manage.py check`
- `docker compose exec web python manage.py test`
