# Task12 Implement

- タスク名: Task12「共通 UI 基盤とレイアウトシェル整備」
- モード: Pencil フロー / implement

## 実装概要

- `shared-ui-foundation_v5` を正本として、共通 `base template`、共通 CSS / JS、共通 flash / form error 表示を実装
- auth shell と signed-in shell を既存画面へ適用
- `Startup Copy Layer` を最小限の JS 付き共通レイヤーとして実装
- home に一時メッセージ領域、主 CTA、日付単位ログ一覧の受け皿を追加
- ボトムナビを `Home / Calendar / Search` の高頻度導線に整理
- 右上ユーティリティ導線を、通知設定 / Export / Import を含む設定ハブとして受け皿実装
