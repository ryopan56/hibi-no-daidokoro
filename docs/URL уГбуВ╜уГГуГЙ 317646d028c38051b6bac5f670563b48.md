| サインアップ実行 | `/signup` | POST | - | 新規登録処理 |
| ホーム | `/home` | GET | ✅ | 歓迎メッセージ＋日付一覧 |
| 今日の記録へ | `/logs/today` | GET | ✅ | 当日記録画面へリダイレクト |

| 検索実行 | `/search` | POST | ✅ | 検索結果表示 |
| 通知設定画面 | `/settings/notifications` | GET | ✅ | 通知設定表示 |
| 通知設定更新 | `/settings/notifications` | POST | ✅ | 通知設定保存 |
| エクスポート画面 | `/export` | GET | ✅ | エクスポート画面 |
