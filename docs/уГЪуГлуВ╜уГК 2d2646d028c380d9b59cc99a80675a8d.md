# ペルソナ

### ある日のこと

平日の夕方から夜にかけて、
仕事や用事を終えて帰宅したあと、
「今日は何にしようか」と台所の前で少し立ち止まる。

料理は嫌いではない。
むしろ、ちゃんと作れた日は気分がいい。
でも毎日となると、
考えることそのものが負担になる日がある。

家族がいる。
自分だけの食事ではないことも多い。
だからこそ、
「ちゃんとできていない気がする」
「楽をしてはいけない気がする」
「美味しく食べて喜んでほしい」
という気持ちが、ない交ぜになっている。

今日は元気か、と聞かれたら、
「まあまあ」と答える。
頑張るほどではないが、投げ出したいほどでもない。

### 別の日のこと

金曜日の夜、
明日は何をしようかと考えている。
この時間が一番の楽しみ。

午前中はちょっと歩いて大きな公園にいこう。
雪も積もってるから娘とそり遊びをしよう。

そのあと何を食べようか。
昼は残った味噌汁とごはん、
それに卵とウィンナーを焼けば十分。

でも、夜はもう少し手の込んだものにしようかな。
公園の帰りに大き目のスーパーに寄ってみよう。
週に一回は魚を食べたい。
この時期はやっぱり真鱈だろう。

スーパーで食材を買い込んで、
クタクタになりながら帰宅。
娘はお昼寝をしている。

起きたら一緒に晩ごはんの準備だ。
この前買ってあげた小さなエプロンをつけるのを楽しみにしているのは
娘だけではない。
